# AI Fitness Nutrition Tracker

An AI-powered nutrition analysis tool built using Amazon PartyRock.

## Overview

This project analyzes a user's body data and food intake to estimate calorie needs and provide personalized nutrition advice.

The system uses prompt engineering to simulate a sports nutritionist that evaluates dietary patterns and provides recommendations.

## Features

- Calorie estimation
- Macronutrient recommendations
- Meal analysis
- Nutritional gap detection
- Optimized meal suggestions

## Inputs

The application accepts:

- Age
- Gender
- Height
- Weight
- Activity level
- Fitness goal
- Daily meals consumed

## Example Output

The AI provides:

- Estimated BMR
- Estimated daily calorie needs
- Macro recommendations
- Meal analysis
- Suggested improvements

## Technologies Used

- Amazon PartyRock
- Amazon Bedrock foundation models
- Prompt engineering
